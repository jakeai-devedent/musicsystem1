
# TuneDial Class - 우리 반 전용 음악 신청 시스템

이 프로젝트는 유튜브 링크를 통해 우리 반만의 실시간 플레이리스트를 만들고 공유하는 서비스입니다.

## 🚀 주요 기능
- **실시간 유튜브 신청**: 유튜브 링크만 넣으면 AI가 제목과 아티스트를 자동으로 분석하여 대기열에 추가합니다.
- **관리자 모드**: 비밀번호(`0605music`)를 입력하면 교실 메인 스피커에서 영상을 재생하고 대기열을 관리(건너뛰기, 삭제)할 수 있습니다.
- **재생 잠금 기능**: 음악 소리가 여러 기기에서 섞이지 않도록, 영상 재생은 관리자 화면에서만 가능합니다.
- **실시간 동기화**: Firebase Firestore를 통해 모든 학생이 동일한 대기열을 실시간으로 확인합니다.

## 📦 깃허브 마이그레이션 가이드
프로젝트 폴더 전체를 올리셔도 포함된 `.gitignore` 파일 덕분에 안전하게 업로드됩니다.

### 포함되는 주요 파일:
- `src/`: 소스 코드 전체
- `docs/backend.json`: 데이터 구조 정의
- `firestore.rules`: 보안 규칙
- `package.json`: 라이브러리 설정

### 제외되는 파일 (자동):
- `node_modules/`: 라이브러리 실제 파일 (용량 과다)
- `.next/`: 빌드 결과물
- `.env`: 환경 변수 (민감 정보)

## 🛠 시작하기
```bash
npm install
npm run dev
```

## 🔐 관리자 정보
- **비밀번호**: `0605music` (대기열 타이틀 옆의 자물쇠 아이콘 클릭 후 입력)
